# Unit 4 Starter

This is a minimal top-level Unit 4 scaffold to keep folder organization consistent with prior units.

## Scope

- This folder is organizational and can host Unit 4-specific experiments.
- If your instructor expects Unit 4 features to extend your existing practice app, implement those features in that app (do not duplicate full app logic here).

## Quick Start

1. Create `.env` from `.env.example`.
2. Install dependencies:
   - `pnpm install`
   - or `npm install`
3. Run in dev mode:
   - `pnpm dev`
   - or `npm run dev`

## Starter Routes

- `GET /` -> simple Unit 4 starter home page.
